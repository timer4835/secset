			3.靶场（虚拟环境，暂时不设计，安全不能保障）
			4.各种开发文档及跳转接口（文章专区，或者各类文章参考，使用说明）
			5.各种POC和EXP（初步想法，包含各种参考文档及漏洞复现）
			6.各种报错及调试经验（用户可以上传）
